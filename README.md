# Engine3d
Graphics Engine for learning purpose. Written in C++ and DirectX11/HLSL.

## Features
- Reflection via Metaprogramming
- Node Tree
- Resource Pool
- Render Queue
- Normal Mapping
- Gauss Filter
- Outline via Stencil Buffer / Highlight Mouse-Picking
